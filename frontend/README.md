# EduGuard Frontend

This is the frontend directory for the EduGuard application. It is built using **React (Vite)**, **Tailwind CSS**, and **Framer Motion**.

For full project documentation, architecture details, and backend setup instructions, please see the [Main Project README](../README.md) located in the root directory.

## Quick Start

```shell
# Install dependencies
npm install

# Run the development server
npm run dev
```
